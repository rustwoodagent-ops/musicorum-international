## Image Guidelines

- Optimize images for web (aim for under 500KB each)
- Use JPG for photographs, PNG for logos
- Recommended dimensions:
  - Hero/portrait images: 1920px wide minimum
  - Gallery images: 800px wide minimum
  - Logo: SVG or PNG with transparent background
